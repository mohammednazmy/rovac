# Documentation Archive

This directory contains documentation that is intentionally preserved but no longer describes the current operating path of the robot.

## What Lives Here

- Earlier architecture plans and exploratory design notes
- Iteration-era wiring references tied to superseded hardware
- Phase summaries and milestone writeups
- Historical review documents

## Archive Structure

| Path | Contents |
|------|----------|
| `architecture/` | Superseded architecture plans and proposals |
| `reviews/` | Historical reviews and assessments |
| `wiring/` | Wiring documents for earlier hardware generations |
| `project_phases/` | Phase and milestone summary documents |

## Policy

- Archived documents are kept for historical context and parts reuse.
- They should not be used as the primary source for bringup, architecture, or runtime behavior.
- Current documentation lives in the top-level `docs/` tree and is expected to match the active repo state.
