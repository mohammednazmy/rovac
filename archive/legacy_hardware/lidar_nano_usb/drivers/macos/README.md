# macOS CH34x driver (offline bundle)

This folder contains the WCH CH34x USB‑serial macOS VCP driver bundle used by `install_ui.py`.

- Bundled file: `CH341SER_MAC.ZIP` (contains `CH34xVCPDriver.pkg`)
- Source page: `https://www.wch-ic.com/downloads/CH34XSER_MAC_ZIP.html`
- Direct download: `https://www.wch-ic.com/download/file?id=334`

If you plan to redistribute this repository or installer, review WCH’s terms for this driver bundle.
